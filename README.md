# Technologies 
Next.js + TailwindCSS

# How to install

1. Clone the project & enter the project directory
```
git clone https://github.com/yegenanil/weatherapp.git && cd weatherapp
```
2. Install dependencies via npm/yarn
```
npm install
```
3. Start the project
```
npm run dev
```
